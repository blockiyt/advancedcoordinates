# AdvancedCoordinates
AdvancedCoordinates is a Coordinate Plugin so display / send or broadcast the executers Location to others Plugin for Minecraft Java Version 1.8-1.16.

# Features

  - broadcast the Location of the Executer to all other Players on the Server
  - display your coordinates to you
  - send your coordinates to an other player


It's powered by the Spigot API


### Installation

AdvancedNTE requires [Spigot](https://www.spigotmc.org/) with a Minecraft Server to run.

Download the Latest Version of the Plugin [here](https://github.com/blockiyt/advancedcoordinates/releases) and put it into your Plugins folder in your Minecraft Server.

Restart or Reload the Server an you are done!

### Commands
- /coords:advancedcoords.coords -> displays the executer the location of himself
- /coords send <player>:advancedcoords.coords.send -> sends the location of the executor to an other player
- /coords broadcast:advancedcoords.coords.broadcast -> brodcast the location of the player to all other players on the Server
